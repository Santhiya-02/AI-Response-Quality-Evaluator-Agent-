1. Local Vector Indexing

Files uploaded into the panel are split into 400-char chunks (80 overlap), embedded using standard local vectors (384-dim dimensions, model: all-MiniLM-L6-v2), and cached in persistent SQLite ChromaDB indexes.

2. Single-File Front-End Design

All UI templates, styles, dynamic script queries, and graphs run from a single zero-dependency HTML interface served straight by FastAPI. No Node.js environment required.

3. Zero-Temperature Agent Prompting

LLM verification tasks execute structured zero-temperature rules using the Llama-3.3-70b-versatile model hosted on Groq Cloud endpoints, forcing deterministic validation responses.

4. Core Technologies
